# Exploratory-Data-Analysis-of-a-Fitness-Dataset
An EDA of a fitness dataset with comprehensive data visualizations and conclusive insights.

## What is this project?
- This project was for my Programming for Analytics class (CIS 372) where I was tasked to do a full EDA of a dataset of my choosing.
- The dataset I chose was chosen from Kaggle. It revolved around fitness data, particularly nutrition info, workouts, demographics, etc.


## What was the result?
- I got many helpful insights such as optimal workout routine aswell as dietary macros to focus on depending on individual's goals
